While loops

While loops brukes når vi ikke vet hvor mange ganger en serie instruksjoner skal gjentas. En while loop gjentas så lenge input-kriteriet er sant. Hver runde sjekkes det og dersom den er det, kjøres koden en gang til.

En while loop har følgende syntaks:
while (condition) {
// code block to be executed
}
  • condition er et uttrykk som resulterer i true/false.

Eksempel:


while (i < 5) {
"The number is " + i;
i++;
}

Denne koden vil printe:
This number is 0
This number is 1
This number is 2
This number is 3
This number is 4
