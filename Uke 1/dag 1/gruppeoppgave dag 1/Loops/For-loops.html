For loops

En for loop har følgende syntaks:

for (statement 1; statement 2; statement 3) {
// code block to be executed
}

  • Statement 1 kjøres én gang før utførelsen av kodeblokken.
    Her initialiseres vanligvis telleren og settes til startverdien.
  • Statement 2 definerer stoppkriteriet for å utføre kodeblokken.
    Her sammenlignes vanligvis telleren mot en gitt verdi.
  • Statement 3 kjøres hver gang etter at kodeblokken har blitt utført.
    Her er det vanlig å oppdatere telleren med den nye verdien.

    • Eksempel:


    for (i = 0; i < 5; i++) {
    "The number is " + i + "<br>";
    }
    Denne koden vil printe:
    This number is 0
    This number is 1
    This number is 2
    This number is 3
    This number is 4
